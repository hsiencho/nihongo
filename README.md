## ✨ Core Features

### 1. 📚 Preview Flashcards
* Interactive flip cards clearly displaying Japanese (Kanji/Kana), Romaji, and Traditional Chinese translations.
* Features color-coded "Part-of-Speech tags" and built-in example sentences.
* Integrates Text-to-Speech (TTS) technology to provide authentic Japanese pronunciation playback.

### 2. 🎯 Interactive Quiz Training
* **🎤 Voice Training:** Utilizes the Web Speech API to listen to your pronunciation and instantly evaluate your speaking accuracy.
* **🎧 Audio Matching:** Listen to the pronunciation and select the corresponding correct vocabulary.
* **👂 Pure Listening Test:** Listen to the Japanese audio and select the correct Chinese meaning.

### 3. 📈 Visual Pitch Accent Indicators
* Automatically generates "dictionary-style" pitch accent visual indicators (using red lines and markers to show high, low, and dropping pitches) to help you with the most accurate pronunciation training.

### 4. ⚙️ Customizable Learning Experience
* Supports font size scaling for an optimal reading experience.
* Customizable "Pass Target" (the number of consecutive correct answers required to master a word).
* Filter vocabulary banks by JLPT level (N5 to N1) or specific themes.

## 🛠️ Tech Stack

1. **Frontend:** HTML5, CSS3, Vanilla JavaScript
2. **API Integration:** Web Speech API (Speech Recognition & Speech Synthesis)
3. **Framework-free Design:** Extremely lightweight. No dependencies or build tools required; it runs directly in the browser.
